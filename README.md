#  Suite Avançada de Análise de Números Primos

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)
[![Status](https://img.shields.io/badge/Status-Active-brightgreen)](https://github.com/seu-usuario/prime-suite)

Uma suite completa e otimizada para análise, geração e visualização de números primos, implementando algoritmos clássicos e modernos com interface interativa.

##  Características Principais

###  **Algoritmos Implementados**
- **Crivo de Eratóstenes** (altamente otimizado)
- **Crivo de Atkin** (corrigido e melhorado)
- **Crivo de Sundaram** (alternativa eficiente)
- **Teste Miller-Rabin** (para números grandes)
- **Pollard's Rho** (fatoração avançada)
- **Fatoração tradicional** (com cache)

###  **Funcionalidades Avançadas**
-  Verificação de primalidade com múltiplos métodos
-  Geração de primos com benchmarks comparativos
-  Fatoração completa com análise estatística
-  Busca por primos gêmeos e clusters
-  Verificação da Conjectura de Goldbach
-  Visualização da Espiral de Ulam
-  Análise de distribuição de primos
-  Sistema de cache multi-nível
-  Comparação com Teorema do Número Primo

###  **Visualizações**
- Espiral de Ulam interativa
- Gráficos de distribuição de primos
- Histogramas de gaps entre primos
- Comparativos de performance
- Análise de densidade prima
